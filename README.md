## Prerequisites
Make sure following software is installed on your PC
- [JDK 17](https://www.oracle.com/java/technologies/downloads/#java17) or later

## Start project

```shell
$ ./mvnw clean package
$ java -jar ./composite-pattern/target/composite-pattern-0.0.1-SNAPSHOT.jar
```
